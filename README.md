# Yolanda Karapetian Personal Homepage

Static GitHub Pages-ready personal website for Yolanda Karapetian.

## Files

- `index.html` - homepage content and structure
- `styles.css` - premium editorial visual system
- `script.js` - scroll reveal, subtle parallax, and background switcher
- `assets/tamira-hero.png` - homepage portrait background
- `assets/background-01.png` - executive wellness hero fallback
- `assets/background-02.png` - boardroom wellness background
- `assets/background-03.png` - studio wellness background

## Personal Photo

The current homepage portrait background is saved at:

```text
assets/tamira-hero.png
```

The homepage uses that image first. If it is not present, it falls back to `assets/background-01.png`.

## GitHub Pages

Upload all files to a GitHub repository, then enable Pages from the repository settings:

1. Settings
2. Pages
3. Build and deployment: Deploy from a branch
4. Branch: `main`
5. Folder: `/root`

The site is fully static and does not require a build step.
